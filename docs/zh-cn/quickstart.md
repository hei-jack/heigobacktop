# 快速开始

HeiGoBackTop的引入方式极其简单，一般情况下，推荐直接下载到项目使用。

使用以下任意方式引入之后， 只需要一句代码就能简单开启使用。

```html
<script src="下载后的路径"></script>
<script>
//script标签不是必要的 具体请根据你的情况取舍
let example = new HeiGoBackTop();
</script>
```
